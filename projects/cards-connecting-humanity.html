Cards Connecting Humanity

A rating app for a conversational card game, built to be passed around a table on one shared device — no accounts, no login. Someone reads a prompt out loud, the group talks, then whoever's holding the device rates it before the deck moves on to the next question.

Open the app → View the code →

How it works:

  • The deck of prompts shuffles fresh each session, with no repeats until it runs out.
  • Each prompt gets rated on two separate 1–5 star scales — Quality and Depth — plus optional free-text feedback.
  • Ratings are sent to a Google Sheet in the background, and queue locally if the device is offline, retrying once it reconnects.
  • Runs entirely as a static site (plain HTML/CSS/JS, no build step) with a Google Apps Script backend — zero server cost.
